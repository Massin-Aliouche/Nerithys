Instructions rapides :

- Ouvrir localement : double-cliquer sur `index.html` ou servir avec un serveur statique.
- Pousser vers GitHub :

```bash
git init
git add .
git commit -m "Initial site prototype"
git remote add origin https://github.com/Massin-Aliouche/Nerithys.git
git branch -M main
git push -u origin main
```

Fichiers principaux : `index.html`, `css/style.css`, `js/main.js`.
